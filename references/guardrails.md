# 营造 · 护栏阈值表（D2 抗过度打磨 · v1.10）

> 阈值集中此处便于岁修调参、不锁死 SKILL 正文。**所有阈值为启发式起点，待 YZ-GUARDRAIL-D2 实测回填**——当前非审计实测。

## G1 候选冗长度上限（`tools/check-bloat.sh`）

- **ρ = 字符比**（`wc -m`，非字节——中文 skill 防 UTF-8 系统性惩罚）= chars(候选正文 + references + templates 净增) / chars(原版同口径)。
- 三档：`ρ≤1.15` 放行 ｜ `1.15<ρ≤1.5` 联动门（须 `realized_gain ≥ Δ/(ρ−1)·θ`）｜ `ρ>1.5` 硬拒重投（落架 `--overhaul` 豁免）。
- **θ 首版 = 1.0**（每膨胀 50% 须偿付 ≥0.5 分 realized_gain）。**为何不是 10**：v1.9 实测 overhaul realized 均值仅 +0.23（0-10 标度），θ=10 会毙掉几乎所有 ρ>1.15 候选、绞杀「技能依赖型高回报区」。

### ρ × 需偿付 Δ 对照（θ=1.0，0-10 标度）

| ρ | 需偿付 realized_gain | 对 +0.23 实测基线的 sanity check |
|---|---|---|
| 1.15 | 0（放行） | — |
| 1.30 | 0.30 | 高于均值 +0.23·须技能依赖型才够 |
| 1.50 | 0.50 | 远高于均值·临界 |
| >1.50 | 硬拒 | 膨胀=需偿付增益的负债 |

## G2 轮数硬上限（兜底·降维）

- `R_max = min(6, |P0|+2)`：兜底防极端长 P0 无限磨。多数 run「连续两轮过不了门」软停先触发，R_max 只兜尾部。**6 = GEPA 早收敛经验类比、非论文数字**。

## G3 held-out 防过拟合

- 命题角色按 `sha1(prompt)%k` 切 `D_gate / D_holdout`，`|D_holdout| ≥ ⌈0.3N⌉ 且 ≥1`；鼓励 N≥4。
- **切分=移植 GEPA Alg1（有 ICLR2026 背书）**；**按 D_holdout 背离 early-stop（D_gate↑ 但 D_holdout 不升连续2轮）= yingzao 原创启发式·无论文背书·待校准**。
- **仅 N≥4 全功能**（切分+过拟合判定+据以 early-stop）；`N<4` 退化为单哨兵（只告警、不构成判定）。
- 落成匾分数以 D_holdout 为准（对冲 v1.9 自评偏置 −Bias=+1.58）。
- eval-harness：`set=="holdout"` 项**不参与门①聚合**、只读报告（CI 防泄漏夹具 holdout-noleak 锁死）。
